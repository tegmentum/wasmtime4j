# Task: Implement Instance lifecycle and import/export management

## Description
Implement WebAssembly instance instantiation with complete import resolution, export access, and lifecycle management. This includes host function binding, memory sharing, and proper cleanup of instance resources.

## Acceptance Criteria
- [ ] Instance creation with import resolution and validation
- [ ] Export function invocation with type checking and parameter conversion
- [ ] Import binding for functions, memory, tables, and globals
- [ ] Instance disposal with proper resource cleanup
- [ ] Thread-safe instance access for concurrent function calls
- [ ] Comprehensive error handling for instantiation and execution failures
- [ ] Support for multiple instances per module

## Technical Details
- Implement `wasmtime::Instance` wrapper with complete lifecycle management
- Create import resolution system supporting functions, memory, tables, globals
- Add export discovery and invocation with proper type validation
- Implement instance-specific memory and resource management
- Create native functions: instance_new, instance_get_export, instance_call_export, instance_dispose
- Add proper parameter marshalling between Java and WebAssembly types
- Implement thread-safe instance access patterns
