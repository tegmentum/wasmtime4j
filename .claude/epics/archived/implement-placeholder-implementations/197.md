## Description

Implement the 6 native methods in JniInstance class that handle WebAssembly instance export discovery and management. This task focuses on finding and accessing exported functions, globals, memories, and tables from instantiated WebAssembly modules.

## Technical Context

The JniInstance class currently has placeholder implementations for export discovery methods. This task will implement the complete functionality for:

### Native Methods to Implement:
1. `nativeGetExport` - Get export by name
2. `nativeGetExports` - Get all exports
3. `nativeGetExportNames` - Get list of export names
4. `nativeHasExport` - Check if export exists
5. `nativeGetExportType` - Get type information for export
6. `nativeClose` - Release native resources
