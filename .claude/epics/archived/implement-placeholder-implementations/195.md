## Description

Implement the 9 native methods in JniFunction class that handle WebAssembly function execution and management. This task focuses on the core function operations including invocation, parameter handling, and result processing.

## Technical Context

The JniFunction class currently has placeholder implementations for all native methods. This task will implement the complete functionality for:

### Native Methods to Implement:
1. `nativeCall` - Execute WebAssembly function with parameters
2. `nativeGetType` - Retrieve function type information

4. `nativeGetResultCount` - Get number of results
5. `nativeGetParamType` - Get specific parameter type
6. `nativeGetResultType` - Get specific result type
7. `nativeAsExtern` - Convert to extern reference
8. `nativeClose` - Release native resources
9. `nativeEquals` - Compare function instances
