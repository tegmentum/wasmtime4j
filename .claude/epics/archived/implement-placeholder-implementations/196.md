## Description

Implement the 13 native methods in JniGlobal class that handle WebAssembly global variable operations. This task focuses on global variable access, mutation, and type management within the WebAssembly runtime.

## Technical Context

The JniGlobal class currently has placeholder implementations for all native methods. This task will implement the complete functionality for:

### Native Methods to Implement:
1. `nativeGet` - Read global variable value
2. `nativeSet` - Write global variable value (if mutable)
3. `nativeGetType` - Get global variable type information
4. `nativeGetValueType` - Get the value type of the global
5. `nativeIsMutable` - Check if global is mutable
6. `nativeAsExtern` - Convert to extern reference
7. `nativeClose` - Release native resources
8. `nativeEquals` - Compare global instances
9. `nativeGetI32` - Get i32 value directly
10. `nativeGetI64` - Get i64 value directly
11. `nativeGetF32` - Get f32 value directly
12. `nativeGetF64` - Get f64 value directly
13. `nativeSetValue` - Set value with type validation
