# Task: Native Library Foundation Extensions

## Description
Implement missing Wasmtime APIs in the wasmtime4j-native Rust library to provide comprehensive native foundation for both JNI and Panama implementations. This includes extending the current native library with new API bindings, proper error handling, resource management, and build system updates to support the complete Wasmtime 36.0.2 API surface.

- [ ] All missing Wasmtime 36.0.2 APIs implemented in wasmtime4j-native/src/lib.rs

- [ ] Memory safety guarantees for all new native functions
- [ ] Updated build.rs to handle new dependencies and compilation requirements
- [ ] Cross-platform compatibility maintained for Linux/Windows/macOS on x86_64 and ARM64
- [ ] Native function exports properly structured for both JNI and Panama consumption
- [ ] Performance optimizations for frequently called native functions
