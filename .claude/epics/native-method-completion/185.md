# Issue #185: Global Variable Operations Implementation

## Overview

Implement complete global variable operations API to enable inspection of global types, reading/writing values, and checking mutability constraints. Essential for WebAssembly programs that use global state management.

## Acceptance Criteria

- [ ] `nativeGetValueType(long globalHandle)` returns global type information
- [ ] `nativeGetValue(long globalHandle)` reads current global value
- [ ] `nativeSetValue(long globalHandle, Object value)` writes global value
- [ ] `nativeIsMutable(long globalHandle)` checks mutability
- [ ] Mutability constraints enforced with type safety
- [ ] Type validation prevents invalid value assignments
- [ ] All methods include defensive parameter validation
- [ ] Integration tests validate global manipulation workflows
