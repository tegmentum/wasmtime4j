# Task: Project Setup and Maven Configuration

## Description

Create a new Maven project `wasmtime4j-native-loader` to handle native library loading, extraction, and platform detection. This project will be a standalone library that can be used by both wasmtime4j-jni and wasmtime4j-panama modules, eliminating code duplication and providing a centralized solution for native library management.

## Acceptance Criteria

- [ ] Create new Maven module `wasmtime4j-native-loader` in project root
- [ ] Configure POM with appropriate dependencies and build configuration
- [ ] Set up proper package structure: `ai.tegmentum.wasmtime4j.nativeloader`
- [ ] Configure Maven build to produce standalone JAR artifact
- [ ] Add module to parent POM reactor build
- [ ] Configure appropriate Java version compatibility (Java 8+)
- [ ] Set up basic logging with java.util.logging
- [ ] Configure static analysis tools (Checkstyle, SpotBugs, Spotless)
- [ ] Create basic module-info.java for Java 9+ compatibility
- [ ] Add appropriate Maven plugins for testing and packaging
