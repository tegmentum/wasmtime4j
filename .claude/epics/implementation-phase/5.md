# Task: Native Library Core - Rust/Wasmtime integration with basic WebAssembly operations

## Description
Implement the foundational native Rust library (`wasmtime4j-native`) that provides unified Wasmtime WebAssembly runtime bindings for both JNI and Panama Foreign Function API implementations. This shared library will serve as the core integration point with Wasmtime, providing essential WebAssembly operations including module compilation, instantiation, function calls, and memory management.

## Acceptance Criteria
- [ ] Rust library project structure created in `wasmtime4j-native/` directory
- [ ] Wasmtime 36.0.2 dependency integrated with Cargo
- [ ] Core WebAssembly operations implemented: Engine creation, Module compilation, Instance creation
- [ ] Memory management functions implemented with proper lifecycle handling
- [ ] Function invocation system implemented for both imported and exported functions
- [ ] Error handling system that maps Wasmtime errors to structured error codes
- [ ] Both JNI and Panama FFI export interfaces defined and implemented
- [ ] Basic resource cleanup and memory safety mechanisms in place
- [ ] Defensive programming measures implemented to prevent JVM crashes

## Technical Details
- **Implementation approach**: Create Rust library using Wasmtime crate with dual export interfaces
- **Key considerations**: 
  - Memory safety and resource lifecycle management
  - Error propagation without JVM crashes
  - Efficient data marshaling between Rust and Java
  - Thread safety for concurrent WebAssembly operations
- **Code locations/files affected**:
  - `wasmtime4j-native/Cargo.toml` - Rust project configuration
  - `wasmtime4j-native/src/lib.rs` - Main library entry point
  - `wasmtime4j-native/src/engine.rs` - Engine management
  - `wasmtime4j-native/src/module.rs` - Module compilation and management
  - `wasmtime4j-native/src/instance.rs` - Instance creation and management
  - `wasmtime4j-native/src/memory.rs` - Memory operations
  - `wasmtime4j-native/src/function.rs` - Function invocation
  - `wasmtime4j-native/src/error.rs` - Error handling and mapping
  - `wasmtime4j-native/src/jni.rs` - JNI export interface
  - `wasmtime4j-native/src/panama.rs` - Panama FFI export interface

## Dependencies
- [ ] Wasmtime 36.0.2 Rust crate
- [ ] JNI Rust crate for JNI bindings
- [ ] Standard Rust toolchain with cross-compilation support
