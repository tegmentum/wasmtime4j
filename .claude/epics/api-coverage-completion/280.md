# Task 280: Factory Method and Runtime Selection Implementation

## Overview

Implement missing factory method logic and runtime selection mechanisms to eliminate ClassNotFoundException scenarios and provide proper Panama → JNI fallback behavior.

## Problem Statement

Analysis revealed that many factory methods have runtime selection comments but incomplete implementations. This creates ClassNotFoundException risks and prevents the unified API from working correctly across different Java versions and runtime availability scenarios.

## Technical Objectives

### 1. Factory Method Implementation
Complete runtime selection logic for:
- `ImportMap.create()` and related factory methods
- `PerformanceProfiler.create()`
- `ResourceUsage.create()`
- `EngineStatistics.create()`
- `CompilationStatistics.create()`
- WASI configuration factories

### 2. Runtime Selection Pattern
Implement consistent pattern:
```java
public static Interface create() {
    try {
        // Try Panama implementation first
        Class<?> panamaClass = Class.forName("ai.tegmentum.wasmtime4j.panama.PanamaImplementation");
        Method createMethod = panamaClass.getDeclaredMethod("create");
        return (Interface) createMethod.invoke(null);
    } catch (ClassNotFoundException | NoSuchMethodException | IllegalAccessException | InvocationTargetException e) {
        try {
            // Fallback to JNI implementation
            Class<?> jniClass = Class.forName("ai.tegmentum.wasmtime4j.jni.JniImplementation");
            Method createMethod = jniClass.getDeclaredMethod("create");
            return (Interface) createMethod.invoke(null);
        } catch (Exception e2) {
            throw new RuntimeException("No wasmtime4j runtime available. Please ensure either wasmtime4j-panama (Java 23+) or wasmtime4j-jni is on the classpath.", e2);
        }
    }
}
```

### 3. Missing Concrete Implementations
Create concrete classes where referenced but missing:
- JNI import map implementations
- Panama import map implementations
- Performance monitoring concrete classes
- WASI configuration concrete classes
