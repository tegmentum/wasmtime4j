# Task 285: Component Model Type System Implementation

## Overview

Implement the WebAssembly Component Model type system to enable interface types, resource management, and component composition. This addresses the 60% coverage gap in Component Model support.

## Problem Statement

Current Component Model implementation provides only 40% coverage with missing core functionality:
- **Interface Types**: Records, variants, lists, and options not implemented
- **Resource Types**: Resource handles and lifetime management missing
- **Type Imports/Exports**: Component interface definitions incomplete
- **Canonical ABI**: WebAssembly Canonical ABI for complex types missing

This prevents use of modern WebAssembly components and WASI Preview 2 modules.

```java
public class ComponentTypeRegistryImpl implements ComponentTypeRegistry {
    private final Map<String, ComponentType> types = new ConcurrentHashMap<>();
    private final Map<String, Set<String>> dependencies = new ConcurrentHashMap<>();

    @Override
    public void registerType(String name, ComponentType type) {
        validateTypeDefinition(name, type);
        types.put(name, type);
        updateDependencies(name, type);
    }

    private void validateTypeDefinition(String name, ComponentType type) {
        // Validate type consistency and detect cycles
        if (detectCycle(name, type)) {
            throw new ComponentException("Circular type dependency detected: " + name);
        }
    }

    private void updateDependencies(String name, ComponentType type) {
        Set<String> typeDeps = extractTypeDependencies(type);
        dependencies.put(name, typeDeps);
    }
}
```

#### Canonical ABI Implementation
```java
public class CanonicalAbi {
    /**
     * Converts Java values to Component Model canonical form.
     */
    public MemorySegment lowerValue(Object value, ComponentType type, Arena arena)
        throws ComponentException {

        return switch (type) {
            case RecordType record -> lowerRecord(value, record, arena);
            case VariantType variant -> lowerVariant(value, variant, arena);
            case ListType list -> lowerList(value, list, arena);
            case OptionType option -> lowerOption(value, option, arena);
            case ResultType result -> lowerResult(value, result, arena);
            default -> throw new ComponentException("Unsupported type: " + type);
        };
    }

    /**
     * Converts Component Model canonical form to Java values.
     */
    public Object liftValue(MemorySegment memory, ComponentType type) throws ComponentException {
        return switch (type) {
            case RecordType record -> liftRecord(memory, record);
            case VariantType variant -> liftVariant(memory, variant);
            case ListType list -> liftList(memory, list);
            case OptionType option -> liftOption(memory, option);
            case ResultType result -> liftResult(memory, result);
            default -> throw new ComponentException("Unsupported type: " + type);
        };
    }
}
```

### Phase 2: Resource Management (Week 1, Days 4-7)

#### Resource Handle Implementation
```java
public class ResourceManagerImpl implements ResourceManager {
    private final AtomicLong nextId = new AtomicLong(1);
    private final Map<Long, ResourceInstance<?>> resources = new ConcurrentHashMap<>();
    private final Map<ResourceType, ResourceDescriptor> descriptors = new ConcurrentHashMap<>();

    @Override
    public <T> ResourceHandle<T> createResource(ResourceType type, T implementation) {
        long id = nextId.getAndIncrement();
        ResourceHandle<T> handle = new ResourceHandleImpl<>(id, type);
        ResourceInstance<T> instance = new ResourceInstance<>(handle, implementation);

        resources.put(id, instance);
        return handle;
    }

    @Override
    public <T> T getResource(ResourceHandle<T> handle, Class<T> type) throws ComponentException {
        ResourceInstance<?> instance = resources.get(handle.getId());
        if (instance == null || !instance.handle().isValid()) {
            throw new ComponentException("Invalid resource handle: " + handle.getId());
        }

        return type.cast(instance.implementation());
    }

    private record ResourceInstance<T>(ResourceHandle<T> handle, T implementation) {}
}
```

#### Resource Lifetime Management
```java
public class ResourceLifecycleManager {
    private final Map<ResourceHandle<?>, LifecycleState> lifecycles = new ConcurrentHashMap<>();

    public void manageResource(ResourceHandle<?> handle, ResourceLifetime lifetime) {
        LifecycleState state = new LifecycleState(lifetime, System.currentTimeMillis());
        lifecycles.put(handle, state);

        // Set up automatic cleanup for owned resources
        if (lifetime == ResourceLifetime.OWNED) {
            scheduleCleanup(handle);
        }
    }

    private void scheduleCleanup(ResourceHandle<?> handle) {
        // Use WeakReference to avoid memory leaks
        WeakReference<ResourceHandle<?>> weakHandle = new WeakReference<>(handle);

        cleanupExecutor.schedule(() -> {
            ResourceHandle<?> ref = weakHandle.get();
            if (ref == null) {
                // Resource was garbage collected, clean up our tracking
                lifecycles.remove(handle);
            }
        }, CLEANUP_DELAY_SECONDS, TimeUnit.SECONDS);
    }

    private record LifecycleState(ResourceLifetime lifetime, long createdAt) {}
}
```

```java
public class ComponentInstantiator {
    private final ResourceManager resourceManager;
    private final CanonicalAbi abi;

    public ComponentInstance instantiate(
        Component component,
        ComponentInterface componentInterface,
        Map<String, Object> imports
    ) throws ComponentException {

        // Validate imports against interface
        validateImports(componentInterface.getImports(), imports);

        // Create resource managers for component resources
        Map<ResourceType, ResourceManager> resourceManagers = createResourceManagers(componentInterface);

        // Link imports and create instance
        return new ComponentInstanceImpl(component, componentInterface, imports, resourceManagers);
    }

    private void validateImports(
        Map<String, ComponentImport> expectedImports,
        Map<String, Object> providedImports
    ) throws ComponentException {

        for (Map.Entry<String, ComponentImport> entry : expectedImports.entrySet()) {
            String name = entry.getKey();
            ComponentImport importDef = entry.getValue();

            if (!providedImports.containsKey(name)) {
                throw new ComponentException("Missing required import: " + name);
            }

            Object provided = providedImports.get(name);
            validateImportType(importDef, provided);
        }
    }
}
```
