# Task 286: Module Serialization and Caching Support

## Overview

Implement module serialization and caching to enable persistent storage of compiled WebAssembly modules, reducing compilation overhead in production environments.

## Problem Statement

Current Module implementation lacks serialization capabilities present in native Wasmtime:
- **Module Serialization**: Cannot persist compiled modules to disk
- **Deserialization**: Cannot load pre-compiled modules from storage
- **Cache Management**: No built-in caching for frequently used modules
- **Cross-Session Persistence**: Module compilation work lost between application restarts

This forces re-compilation of modules on every application startup, impacting cold start performance.

## Technical Objectives

### 1. Module Serialization API

#### Core Serialization Interface
```java
public interface Module extends Closeable {
    /**
     * Serializes this compiled module to bytes.
     *
     * <p>The serialized bytes can be stored persistently and later
     * deserialized to recreate the module without recompilation.
     *
     * @return serialized module bytes
     * @throws WasmException if serialization fails
     */
    byte[] serialize() throws WasmException;

    /**
     * Deserializes a module from previously serialized bytes.
     *
     * @param engine the engine to associate with the module
     * @param serializedBytes bytes from previous serialize() call
     * @return deserialized Module instance
     * @throws WasmException if deserialization fails or incompatible format
     */
    static Module deserialize(Engine engine, byte[] serializedBytes) throws WasmException;

    /**
     * Checks if bytes represent a serialized module.
     *
     * @param bytes the bytes to examine
     * @return true if bytes appear to be a serialized module
     */
    static boolean isSerializedModule(byte[] bytes);

    /**
     * Gets metadata about this module for caching decisions.
     *
     * @return module metadata including compilation timestamp, engine config hash
     */
    ModuleMetadata getMetadata();
}
```

#### Module Metadata for Cache Management

```java
    long hitCount,
    long missCount,
    long evictionCount,
    long totalSize,

        long total = hitCount + missCount;
```

#### Cache Key Strategy
```java
public class ModuleCacheKey {
    private final String wasmBytesHash;
    private final String engineConfigHash;
    private final String compilationOptions;

    public static ModuleCacheKey create(byte[] wasmBytes, Engine engine) {
        String wasmHash = computeHash(wasmBytes);
        String engineHash = engine.getConfig().getConfigHash();
        String options = serializeCompilationOptions(engine.getConfig());

        return new ModuleCacheKey(wasmHash, engineHash, options);
    }

    public String toStringKey() {
        return String.format("%s:%s:%s", wasmBytesHash, engineConfigHash, compilationOptions);
    }

    private static String computeHash(byte[] bytes) {
        MessageDigest digest = MessageDigest.getInstance("SHA-256");
        byte[] hash = digest.digest(bytes);
        return Base64.getEncoder().encodeToString(hash);
    }
}
```

### 3. File-Based Cache Implementation

#### Persistent Disk Cache
```java
public class DiskModuleCache implements ModuleCache {
    private final Path cacheDirectory;
    private final long maxCacheSize;
    private final Duration entryTtl;
    private final CacheStatistics statistics = new CacheStatistics();

    public DiskModuleCache(Path cacheDirectory, long maxCacheSize, Duration entryTtl) {
        this.cacheDirectory = cacheDirectory;
        this.maxCacheSize = maxCacheSize;
        this.entryTtl = entryTtl;

        ensureCacheDirectory();
        loadExistingEntries();
    }

    @Override
    public void put(String key, Module module) throws CacheException {
        try {
            Path entryPath = getEntryPath(key);
            ModuleCacheEntry entry = new ModuleCacheEntry(
                module.getMetadata(),
                module.serialize(),
                System.currentTimeMillis()
            );

            Files.write(entryPath, serializeEntry(entry));
            statistics.recordPut(entry.serializedModule().length);

            // Trigger cleanup if cache is full
            if (getCurrentCacheSize() > maxCacheSize) {
                performMaintenance();
            }

        } catch (Exception e) {
            throw new CacheException("Failed to cache module: " + key, e);
        }
    }

    @Override
    public Optional<Module> get(String key, Engine engine) throws CacheException {
        try {
            Path entryPath = getEntryPath(key);
            if (!Files.exists(entryPath)) {
                statistics.recordMiss();
                return Optional.empty();
            }

            ModuleCacheEntry entry = deserializeEntry(Files.readAllBytes(entryPath));

            // Check if entry is expired
            if (isExpired(entry)) {
                Files.deleteIfExists(entryPath);
                statistics.recordMiss();
                return Optional.empty();
            }

            // Check if entry is compatible with current engine
            if (!entry.metadata().isCompatibleWith(engine)) {
                statistics.recordMiss();
                return Optional.empty();
            }

            Module module = Module.deserialize(engine, entry.serializedModule());
            statistics.recordHit();
            return Optional.of(module);

        } catch (Exception e) {
            statistics.recordMiss();
            throw new CacheException("Failed to retrieve cached module: " + key, e);
        }
    }

    private record ModuleCacheEntry(
        ModuleMetadata metadata,
        byte[] serializedModule,
        long cachedAt
    ) {}
}
```

### 4. Memory-Based Cache Implementation

#### In-Memory LRU Cache
```java
public class MemoryModuleCache implements ModuleCache {
    private final int maxEntries;
    private final Map<String, CachedModule> cache;
    private final CacheStatistics statistics = new CacheStatistics();

    public MemoryModuleCache(int maxEntries) {
        this.maxEntries = maxEntries;
        this.cache = new LinkedHashMap<String, CachedModule>(16, 0.75f, true) {
            @Override
            protected boolean removeEldestEntry(Map.Entry<String, CachedModule> eldest) {
                if (size() > maxEntries) {
                    statistics.recordEviction();
                    return true;
                }
                return false;
            }
        };
    }

    @Override
    public synchronized void put(String key, Module module) throws CacheException {
        try {
            byte[] serialized = module.serialize();
            CachedModule cached = new CachedModule(
                module.getMetadata(),
                serialized,
                System.currentTimeMillis()
            );

            cache.put(key, cached);
            statistics.recordPut(serialized.length);

        } catch (Exception e) {
            throw new CacheException("Failed to cache module in memory: " + key, e);
        }
    }

    @Override
    public synchronized Optional<Module> get(String key, Engine engine) throws CacheException {
        CachedModule cached = cache.get(key);

        if (cached == null) {
            statistics.recordMiss();
            return Optional.empty();
        }

        if (!cached.metadata().isCompatibleWith(engine)) {
            cache.remove(key); // Remove incompatible entry
            statistics.recordMiss();
            return Optional.empty();
        }

        try {
            Module module = Module.deserialize(engine, cached.serializedBytes());
            statistics.recordHit();
            return Optional.of(module);

        } catch (Exception e) {
            cache.remove(key); // Remove corrupted entry
            statistics.recordMiss();
            throw new CacheException("Failed to deserialize cached module: " + key, e);
        }
    }

    private record CachedModule(
        ModuleMetadata metadata,
        byte[] serializedBytes,
        long cachedAt
    ) {}
}
```

### Phase 1: Serialization Infrastructure (Days 1-3)

#### Native Serialization Support
```c
// JNI implementation for module serialization
JNIEXPORT jbyteArray JNICALL Java_Module_serializeNative(
    JNIEnv *env, jobject obj, jlong module_ptr
) {
    wasmtime_module_t *module = (wasmtime_module_t*)module_ptr;

    wasm_byte_vec_t serialized;
    wasmtime_error_t *error = wasmtime_module_serialize(module, &serialized);

    if (error != NULL) {
        wasmtime_error_message(error, &error_message);
        (*env)->ThrowNew(env, wasmException, wasm_byte_vec_data(&error_message));
        wasmtime_error_delete(error);
        return NULL;
    }

    jbyteArray result = (*env)->NewByteArray(env, serialized.size);
    (*env)->SetByteArrayRegion(env, result, 0, serialized.size, (jbyte*)serialized.data);

    wasm_byte_vec_delete(&serialized);
    return result;
}

JNIEXPORT jlong JNICALL Java_Module_deserializeNative(
    JNIEnv *env, jclass clazz, jlong engine_ptr, jbyteArray serialized_bytes
) {
    wasmtime_engine_t *engine = (wasmtime_engine_t*)engine_ptr;

    jsize len = (*env)->GetArrayLength(env, serialized_bytes);
    jbyte *bytes = (*env)->GetByteArrayElements(env, serialized_bytes, NULL);

    wasmtime_module_t *module;
    wasmtime_error_t *error = wasmtime_module_deserialize(
        engine, (uint8_t*)bytes, len, &module);

    (*env)->ReleaseByteArrayElements(env, serialized_bytes, bytes, JNI_ABORT);

    if (error != NULL) {
        wasmtime_error_message(error, &error_message);
        (*env)->ThrowNew(env, wasmException, wasm_byte_vec_data(&error_message));
        wasmtime_error_delete(error);
        return 0;
    }

    return (jlong)module;
}
```

#### Panama Implementation
```java
public class PanamaModuleImpl implements Module {
    private final MemorySegment moduleHandle;

    @Override
    public byte[] serialize() throws WasmException {
        try (Arena arena = Arena.ofConfined()) {
            MemorySegment sizeSegment = arena.allocate(JAVA_LONG);

            int result = WasmtimeNative.wasmtime_module_serialize(
                moduleHandle, MemorySegment.NULL, sizeSegment);

            if (result != 0) {
                throw new WasmException("Failed to get serialized size");
            }

            long size = sizeSegment.get(JAVA_LONG, 0);
            MemorySegment buffer = arena.allocateArray(JAVA_BYTE, size);

            result = WasmtimeNative.wasmtime_module_serialize(moduleHandle, buffer, sizeSegment);

            if (result != 0) {
                throw new WasmException("Failed to serialize module");
            }

            return buffer.toArray(JAVA_BYTE);
        }
    }

    public static Module deserialize(Engine engine, byte[] serializedBytes) throws WasmException {
        try (Arena arena = Arena.ofConfined()) {
            MemorySegment engineHandle = ((PanamaEngineImpl) engine).getHandle();
            MemorySegment bytesSegment = arena.allocateArray(JAVA_BYTE, serializedBytes);
            MemorySegment moduleHandlePtr = arena.allocate(ADDRESS);

            int result = WasmtimeNative.wasmtime_module_deserialize(
                engineHandle, bytesSegment, serializedBytes.length, moduleHandlePtr);

            if (result != 0) {
                throw new WasmException("Failed to deserialize module");
            }

            MemorySegment moduleHandle = moduleHandlePtr.get(ADDRESS, 0);
            return new PanamaModuleImpl(moduleHandle);
        }
    }
}
```

### Phase 2: Cache Implementation (Days 4-5)

#### Unified Cache Interface
```java
public class ModuleCacheFactory {
    public static ModuleCache createMemoryCache(int maxEntries) {
        return new MemoryModuleCache(maxEntries);
    }

    public static ModuleCache createDiskCache(Path cacheDir, long maxSize, Duration ttl) {
        return new DiskModuleCache(cacheDir, maxSize, ttl);
    }

    public static ModuleCache createTieredCache(ModuleCache l1, ModuleCache l2) {
        return new TieredModuleCache(l1, l2);
    }
}

public class TieredModuleCache implements ModuleCache {
    private final ModuleCache l1Cache; // Memory cache
    private final ModuleCache l2Cache; // Disk cache

    @Override
    public Optional<Module> get(String key, Engine engine) throws CacheException {
        // Try L1 cache first
        Optional<Module> module = l1Cache.get(key, engine);
        if (module.isPresent()) {
            return module;
        }

        // Try L2 cache
        module = l2Cache.get(key, engine);
        if (module.isPresent()) {
            // Promote to L1 cache
            l1Cache.put(key, module.get());
        }

        return module;
    }

    @Override
    public void put(String key, Module module) throws CacheException {
        // Store in both caches
        l1Cache.put(key, module);
        l2Cache.put(key, module);
    }
}
```

### Phase 3: Integration and Optimization (Days 6-7)

#### Engine Integration
```java
public class EngineImpl implements Engine {
    private final ModuleCache moduleCache;

    @Override
    public Module compileModule(byte[] wasmBytes) throws WasmException {
        if (moduleCache != null) {
            ModuleCacheKey key = ModuleCacheKey.create(wasmBytes, this);
            Optional<Module> cached = moduleCache.get(key.toStringKey(), this);

            if (cached.isPresent()) {
                return cached.get();
            }
        }

        // Compile module normally
        Module module = compileModuleInternal(wasmBytes);

        // Cache the compiled module
        if (moduleCache != null) {
            ModuleCacheKey key = ModuleCacheKey.create(wasmBytes, this);
            moduleCache.put(key.toStringKey(), module);
        }

        return module;
    }
}
```

## Performance Requirements

- **Serialization**: <100ms for modules up to 10MB
- **Deserialization**: 3x+ faster than compilation from source
- **Cache Hit**: <1ms lookup time for memory cache
- **Cache Storage**: <10% overhead vs original module size
- **Memory Usage**: <1MB overhead per 100 cached modules
