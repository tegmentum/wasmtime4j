# Wasmtime4j API Reference

## Core APIs

### Engine
The Engine is the compilation context for WebAssembly modules.

```java
// Create an engine with default configuration
Engine engine = Engine.create();

// Create an engine with custom configuration
EngineConfig config = EngineConfig.builder()
    .withOptimizationLevel(OptimizationLevel.SPEED)
    .withParallelCompilation(true)
    .build();
Engine engine = Engine.create(config);
```

**Thread Safety**: Engines are thread-safe and can be shared across threads.
**Resource Management**: Always close engines when done or use try-with-resources.
**Performance**: Reuse engines when possible as creation is expensive.
```

### 1. Load and Execute a WebAssembly Module

```java
import ai.tegmentum.wasmtime4j.*;

public class QuickStart {
    public static void main(String[] args) {
        // Load WASM bytes
        byte[] wasmBytes = Files.readAllBytes(Paths.get("example.wasm"));

        try (Engine engine = Engine.create();
             Store store = Store.create(engine);
             Module module = Module.compile(engine, wasmBytes);
             Instance instance = Instance.create(store, module)) {

            // Get exported function
            WasmFunction add = instance.getExport("add");

            // Call the function
            Object[] result = add.call(5, 3);
            System.out.println("Result: " + result[0]); // 8
        }
    }
}

```java
// examples/src/main/java/examples/
public class AdvancedExamples {

    /**
     * Example: Web Service Plugin System
     * Demonstrates loading WebAssembly plugins for HTTP request processing
     */
    public void webServicePlugin() throws Exception {
        byte[] pluginWasm = loadWebServicePlugin();

        try (Engine engine = Engine.create(EngineConfig.forProduction());
             Store store = Store.create(engine);
             Module module = Module.compile(engine, pluginWasm)) {

            // Create linker for host functions
            Linker linker = Linker.create(engine);

            // Define host functions available to plugins
            linker.defineHostFunction("env", "log", (params) -> {
                System.out.println("Plugin log: " + params[0]);
                return new Object[0];
            });

            linker.defineHostFunction("env", "http_response", (params) -> {
                int status = (Integer) params[0];
                String body = (String) params[1];
                sendHttpResponse(status, body);
                return new Object[0];
            });

            // Instantiate plugin
            Instance instance = linker.instantiate(store, module);

            // Handle HTTP request through plugin
            WasmFunction handleRequest = instance.getExport("handle_request");
            handleRequest.call("GET", "/api/users", "{}");
        }
    }

    /**
     * Example: Data Processing Pipeline
     * Demonstrates component model for data processing
     */
    public void dataProcessingPipeline() throws Exception {
        // Load processing stages
        Component filterComponent = Component.load(loadFilterWasm());
        Component transformComponent = Component.load(loadTransformWasm());
        Component aggregateComponent = Component.load(loadAggregateWasm());

        try (Engine engine = Engine.create();
             Store store = Store.create(engine)) {

            // Create component linker
            ComponentLinker linker = ComponentLinker.create(engine);

            // Link components in pipeline
            ComponentInstance filter = linker.instantiate(store, filterComponent);
            ComponentInstance transform = linker.instantiate(store, transformComponent);
            ComponentInstance aggregate = linker.instantiate(store, aggregateComponent);

            // Process data through pipeline
            byte[] inputData = loadInputData();
            byte[] filtered = filter.call("process", inputData);
            byte[] transformed = transform.call("process", filtered);
            byte[] result = aggregate.call("process", transformed);

            saveProcessedData(result);
        }
    }

    /**
     * Example: Async/Reactive Processing
     * Demonstrates async and reactive APIs
     */
    public void asyncProcessing() throws Exception {
        byte[] wasmBytes = loadAsyncProcessor();

        try (Engine engine = Engine.create();
             Store store = Store.create(engine)) {

            // Compile module asynchronously
            CompletableFuture<Module> moduleF = AsyncModule.compileAsync(engine, wasmBytes);

            // Chain async operations
            CompletableFuture<String> resultF = moduleF
                .thenCompose(module -> {
                    Instance instance = Instance.create(store, module);
                    AsyncFunction processor = instance.getExportAsync("process_data");
                    return processor.callAsync("large dataset");
                })
                .thenApply(result -> (String) result[0]);

            // Wait for completion with timeout
            String result = resultF.get(30, TimeUnit.SECONDS);
            System.out.println("Processed: " + result);
        }
    }
}
```

// wasmtime4j-test-utils/src/main/java/
public class WasmTestUtils {

    /**
     * Creates a test engine with optimal settings for testing
     */
    public static Engine createTestEngine() {
        return Engine.create(EngineConfig.builder()
            .withOptimizationLevel(OptimizationLevel.NONE) // Fast compilation
            .withDebugInfo(true) // Enable debugging
            .withParallelCompilation(false) // Deterministic
            .build());
    }

    /**
     * Generates a simple WebAssembly module for testing
     */
    public static byte[] generateTestModule(String functionName, WasmType... paramTypes) {
        WasmModuleBuilder builder = new WasmModuleBuilder();
        builder.addFunction(functionName, paramTypes, WasmType.I32)
            .addBody(builder -> {
                // Simple function that returns sum of parameters
                for (int i = 0; i < paramTypes.length; i++) {
                    builder.local_get(i);
                    if (i > 0) builder.i32_add();
                }
            })
            .export();

        return builder.build();
    }

    /**
     * Asserts that two implementations produce identical results
     */
    public static void assertImplementationParity(
            Supplier<Object[]> jniCall,
            Supplier<Object[]> panamaCall) {

        Object[] jniResult = jniCall.get();
        Object[] panamaResult = panamaCall.get();

        assertThat(jniResult).isEqualTo(panamaResult);
    }
}

public class WasmBenchmarkUtils {

    @BenchmarkMode(Mode.AverageTime)
    @OutputTimeUnit(TimeUnit.MICROSECONDS)
    public static class ModuleCompilationBenchmark {

        @Param({"JNI", "Panama"})
        private String runtime;

        @Benchmark
        public Module compileModule(ModuleSource source) {
            Engine engine = createEngine(runtime);
            return Module.compile(engine, source.getBytes());
        }
    }

    public static BenchmarkResult runCompilationBenchmark(byte[] wasmBytes) {
        Options opt = new OptionsBuilder()
            .include(ModuleCompilationBenchmark.class.getSimpleName())
            .param("wasmBytes", Base64.getEncoder().encodeToString(wasmBytes))
            .build();

        Collection<RunResult> results = new Runner(opt).run();
        return BenchmarkResult.from(results);
    }
}

### Migration and Upgrade Guides

#### 1. **Migration from Other WebAssembly Runtimes**
```markdown
# Migration Guide: From wasmtime-java to wasmtime4j

## Key Differences

### Engine Creation
**Before (wasmtime-java):**
```java
Engine engine = new Engine();
```

**After (wasmtime4j):**
```java
Engine engine = Engine.create();
// Or with configuration
Engine engine = Engine.create(EngineConfig.forProduction());
```

### Module Compilation
**Before:**
```java
Module module = Module.fromBinary(engine, wasmBytes);
```

**After:**
```java
Module module = Module.compile(engine, wasmBytes);
```

### Function Calls
**Before:**
```java
int result = func.call(store, 5, 3)[0].i32();
```

**After:**
```java
Object[] results = func.call(5, 3);
int result = (Integer) results[0];
```
```
